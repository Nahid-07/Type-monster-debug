Event loop helps us to asynchonus our code. For the event loop system JavaScript type one of thread language can work asynchronusly. we know that js is interpreted language mrans it execute codes line by line. suppose we have a heavy weigth code in a function which will take more time to execute.In this other code will wait for execution.Oviously this system is not convenient. For this use case event loop were created.The Event Loop has one simple job — to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop
